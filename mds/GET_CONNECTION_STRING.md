# Get Supabase Connection String - Step by Step

## Open This Link

**Click here:** https://supabase.com/dashboard/project/ntvdjfwmkscuzqkvtqic/settings/database

## Follow These Steps:

### Step 1: Scroll Down

Scroll to find the section titled **"Connection string"**

### Step 2: Find the Dropdown

You'll see a dropdown menu that says something like:

- "Connection string"
- "Connection pooling"
- "Session mode"
- Or similar

### Step 3: Select Connection Pooling

Click on the dropdown and select **"Connection pooling"**

### Step 4: Select Transaction Mode

When you select "Connection pooling", you'll see two options:

- **Session mode**
- **Transaction mode**

**Select "Transaction mode"**

### Step 5: Copy the Connection String

You'll see a text field with a connection string that looks like:

```
postgresql://postgres.ntvdjfwmkscuzqkvtqic:G9KlMVmgu1HYNkBo@aws-1-eu-west-1.pooler.supabase.com:6543/postgres?pgbouncer=true
```

**Click the "Copy" button** next to it.

### Step 6: Paste It Here

Come back here and paste the connection string you copied.
